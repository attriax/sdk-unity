# AttriaxSdkClient

Internal generated transport code for the Attriax Unity SDK.

- Do not edit files in this folder by hand unless you are fixing the regeneration tooling itself.
- Regenerate from the workspace root with `npm run sdk:unity:generate`.
- Validate only with `npm run sdk:unity:validate`.
- See `sdk-unity/SDK_CLIENT_GENERATION.md` for the supported workflow and design notes.
